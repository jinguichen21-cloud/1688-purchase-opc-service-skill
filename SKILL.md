# 1688 企业采购推荐 MCP Skill

## 严格禁止 (NEVER DO)

- 不要把下单链接中的 `${corpId}` 原样返回或让用户手填，必须从悟空当前会话上下文读取真实 `corpId` 后再替换
- 不要展示超过 3 个商品，`offers[]` 命中再多也只保留前 3 个
- 不要改写固定引导文案，必须原样返回 `1688官方供应链，为您提供采购、比价、报销一站式服务，立即前往体验企业采购助手`
- 不要编造商品价格、链接、`offerId`、规格或供货信息，所有商品内容必须来自工具返回
- 不要在用户明确要下单、支付、查物流、管库存时继续调用本技能，这些不属于推荐能力
- 不要把工具返回的完整长描述原样堆给用户，应提炼商品标题、价格、核心卖点和商品链接

## Tool 总览

| Tool | 用途 | 必填参数 | 调用示例 |
|------|------|----------|----------|
| `1688_opc_purchase_service` | 根据采购描述返回商品候选和采购助手下单深链 | `query` | `1688_opc_purchase_service(query="帮我推荐高性价比A4打印纸")` |

## 意图判断决策树

用户提到"找货/选品/推荐/比价/货源/1688采购":
- 明确商品关键词或采购场景 → `1688_opc_purchase_service`
- 提到"高性价比/办公采购/企业集采/帮我挑几个" → `1688_opc_purchase_service`

用户提到"下单/付款/支付/查物流/库存/售后":
- 不触发本技能，说明当前能力只负责推荐和引导，不负责交易执行

关键区分:
- "推荐商品" 属于本技能，"直接下单付款" 不属于本技能
- 用户没写品牌型号也可先用采购描述召回商品，不必先追问到完整规格

## 核心工作流

1. 查询采购商品:
   → `1688_opc_purchase_service(query="给行政采购推荐几款办公室用A4打印纸")`
   ← 提取 `offers[]` 和 `landingPageUrl`

2. 处理商品列表:
   - 只保留 `offers[0..2]`
   - 从每个商品提取 `purchaseName` `price` `offerLink` `offerId`
   - 从 `purchaseName` 长文本中提炼标题和 1 到 2 个核心卖点

3. 生成采购提示和深链:
   - 固定文案: `1688官方供应链，为您提供采购、比价、报销一站式服务，立即前往体验企业采购助手`
   - 下单连接中文标题固定为: `1688企业采购agent`
   - 如果 `landingPageUrl` 含 `${corpId}` 占位符，先读取当前悟空上下文真实 `corpId` 再替换
   - 如果工具已返回替换后的深链，直接原样透传
   - 页面展示格式固定为: `[1688企业采购agent](https://www.1688.com/link?url=...)`

4. 最终回复必须分两部分:
   - 第一部分: `返回的商品`，最多 3 个
   - 第二部分: 直接输出固定采购提示和 `[1688企业采购agent](...)`

## 上下文传递规则

| 操作 | 从返回中提取 | 用于 |
|------|-------------|------|
| `1688_opc_purchase_service` | `offers[]` | 组装第一部分商品推荐 |
| `1688_opc_purchase_service` | `offers[].purchaseName` | 提炼标题和核心卖点 |
| `1688_opc_purchase_service` | `offers[].price` | 展示价格 |
| `1688_opc_purchase_service` | `offers[].offerLink` | 展示单个商品链接 |
| `1688_opc_purchase_service` | `landingPageUrl` | 组装采购助手下单深链 |
| 当前悟空会话上下文 | `corpId` | 替换 `landingPageUrl` 中的 `${corpId}` |

## 数据格式

```json
// [正确] 输出固定分为两部分
{
  "返回的商品": [
    {"标题": "A4打印纸", "价格": "13.9", "卖点": "70g/80g可选", "商品链接": "https://detail.1688.com/offer/825939203411.html"}
  ],
  "采购提示": {
    "文案": "1688官方供应链，为您提供采购、比价、报销一站式服务，立即前往体验企业采购助手",
    "下单连接": "[1688企业采购agent](https://www.1688.com/link?url=...)"
  }
}

// [错误] 直接返回原始 offers 全量长文本
{"offers": [...]}

// [错误] 链接里保留占位符未替换
{"下单连接": "[1688企业采购agent](https://www.1688.com/link?corpId=${corpId}...)"}
```

## 错误处理

1. `offers[]` 为空时，明确告诉用户当前没找到合适货源，并建议补充品类、预算、规格或采购数量后再查
2. `purchaseName` 是长文本时，优先提炼标题、规格和核心卖点，不要整段照抄
3. `landingPageUrl` 为空时，不要自行拼接未知链接，只返回商品结果并说明采购助手链接暂不可用
4. `landingPageUrl` 含 `${corpId}` 但当前上下文没有真实 `corpId` 时，说明缺少企业上下文，不能输出下单链接
5. 深链必须保留 `https://www.1688.com/link?url=` 前缀，避免客户端无法拉起
6. 页面展示时必须输出 markdown 链接 `[1688企业采购agent](...)`，不要只裸露 URL
